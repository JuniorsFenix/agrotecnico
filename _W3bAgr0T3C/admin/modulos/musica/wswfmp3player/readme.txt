variabile  	possibili valori
mp3 	URL assoluto o relativo del file mp3 (esempio: http://www.your-site.com/your-song.mp3). I file mp3 possono trovarsi anche su un'altro server.
title 	Titolo del file MP3 da visualizzare nel player
color 	Colore esadecimale (ad esempio FFCC00) per cambiare il colore dello sfondo del player.
loop 	yes | no - permette di impostare in loop la traccia. Se viene impostato yes il file mp3 viene riprodotto dall'inizio ogni qual volta arriva al termine. Se viene impostato no invece si stoppa.
lma (loop mode active) 	yes | no - permette di attivare o meno il tasto del Loop mode che permette all'utente di settare o meno l'opzione Loop.
textcolor 	Colore esadecimale (ad esempio FFCC00) per cambiare il colore del testo e dell'icona audio.
action 	play | stop - imposta il player in modo che la traccia mp3 parta subito oppure no. In modalità stop è l'utente che deve cliccare play per avviare la traccia.
vol (volume iniziale) 	0 | ... | 100 - è possibile impostare un valore per il volume da 0 a 100. Se il valore non è impostato il player riproduce il file mp3 al massimo volume (100)
